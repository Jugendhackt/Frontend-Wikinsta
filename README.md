# Wik*i*nsta

## Das Endprodukt

- Instagram welche ausschließlich Wissen vermittelt
  -> Wissen kurz verfasst (Wikipedia in küerzer)

## Problem/Fragestellung

- zu wenig Allgemeinwissen(bei Jugendlichen)
- kurze Aufmerksamkeitsspanne
